|   ├─ sites ------------------------------------ 虚拟主机配置文件目录
|   |  ├─ tls ------------------------- tls证书相关

|   |  |
|   |  ├─ redis ----------------------- redis
|   |  |  ├─ config --------- 配置目录
|   |  |  |  ├─ custom 自定义模块配置目录
|   |  |  |  ├─ source.conf 源配置文件
|   |  |  |  └─ ...
|   |  |  ├─ tls ------------ tls证书相关
|   |  |  └─ ...
|   |  |
|   |  ├─ postgres -------------------- postgres

|   |  |  └─ ...
|   |  |
|   |
