# 纯手工搭建一个完善的 PHP 环境

php 开发/部署环境，主要涉及 `SQLite3`、`Redis`、`PostgreSQL`、`PHP`、`Nginx`、`MySQL`

## 使用

```bash
# 下载 node 包
pnpm install
# 开发预览
pnpm dev
# 打包构建
pnpm build
# 打包完成后测试站点
pnpm preview
```
